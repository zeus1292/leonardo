# Tag Registry

Tags used across tasks, projects, and notes for easy filtering.

| Tag | Meaning |
|-----|---------|
| `#job` | Job search related |
| `#idea` | Product idea / validation |
| `#build` | Active building / coding |
| `#portfolio` | Portfolio / personal brand |
| `#research` | Research and learning |
| `#os` | Personal OS setup and maintenance |
| `#meeting` | Meeting prep or follow-up |
| `#writing` | Writing, articles, LinkedIn posts |
| `#admin` | Administrative tasks |
